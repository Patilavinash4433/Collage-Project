HTML

- HTML is the standard markup language for creating Web pages.

- Creator : Tim Berners-Lee


- First Released : 1993


- Usage : Web development, to create webpages, emails, ebooks, and more...


- Syntax and semantics :


  • HTML consists of a series of elements.

  • HTML stands for Hyper Text Markup Language.

  • HTML describes the structure of a Web page.

  • HTML elements tell the browser how to display the content.

  • HTML is the standard markup language for creating Web pages.

  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc..

  • To create a function in JavaScript there are different ways such as a function keyword, arrow function, function expression, and callback function.

    • Python

    - Python is a popular programming language.

    - Creator : Guido van Rossum


    - First Released : 1991


    - Usage : Web development, Data science and data analysis, Mobile app development, artificial intelligence, scientific computing.


    - Syntax and semantics :


  • Python is meant to be an easily readable language.

  • It has fewer syntactic exceptions and special cases than C

  • Its formatting is visually uncluttered and often uses English keywords where other languages use punctuation.

  • Python was designed for readability, and has some similarities to the English language with influence from mathematics.

  • Python uses new lines to complete a command, as opposed to other programming languages which often use semicolons or parentheses.

  • Unlike many other languages, it does not use curly brackets to delimit blocks, and semicolons after statements are allowed but rarely used.

  • Python relies on indentation, using whitespace, to define scope; such as the scope of loops, functions and classes. Other programming languages often use curly-brackets for this purpose.

    • JavaScript

    - A versatile scripting language primarily used for web development.

    - Creator : Brendan Eich


    - First Released : 1995


    - Usage : Web development, Mobile app development, server-side scripting, Communicating with backend servers, game development, Blockchain development.


    - Syntax and semantics :


  • JavaScript variables can be created using var, let, and const keywords.

  • A String in JavaScript can be a written in single quotes or double quotes.

  • To assign a value to a variable in JavaScript we can use the equal to operator.

  • The if-else or loops statements in JavaScript are written inside curly braces blocks.

  • Variables names in JavaScript are case-sensitive and we cannot use the reserved keywords.

  • The semicolon is not necessary for JavaScript at the end of the statement as in other languages.

  • To create a function in JavaScript there are different ways such as a function keyword, arrow function, function expression, and callback function.

    • Java

    - A widely-used object-oriented programming language designed for portability.

    - Creator : James Gosling


    - First Released : 1995


    - Usage : Web applications, mobile applications, enterprise software.


    - Syntax and semantics :


  • The name of the java file must match the class name.

  • Java's syntax is based on C and C++, but it has some unique features.

  • Java is case-sensitive: "MyClass" and "myclass" has different meaning.

  • For example, Java uses the dot operator (.) to access data fields and methods.

  • When saving the file, save it using the class name and add ".java" to the end of the filename.

  • To run the example above on your computer, make sure that Java is properly installed: Go to the Get Started Chapter for how to install Java.

    -First Released : 1994


    -Usage : Web development, Database applications , Desktop applications, Database management, Other uses.

    • C++

    A modern, object-oriented programming language developed.

    Creator: Bjarne Stroustrup


    First Released: 1979


    Usage: Web development, game development, enterprise applications.


    -Syntax and semantics :


  • C++ is one of the world's most popular programming languages.

  • C++ can be found in today's operating systems, Graphical User Interfaces, and embedded systems.

  • C++ is an object-oriented programming language which gives a clear structure to programs and allows code to be reused, lowering development costs.

  • C++ is portable and can be used to develop applications that can be adapted to multiple platforms.

  • C++ is fun and easy to learn!

  • As C++ is close to C, C# and Java, it makes it easy for programmers to switch to C++ or vice versa.
